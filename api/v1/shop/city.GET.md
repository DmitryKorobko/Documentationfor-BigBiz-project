Список городов
==============

`GET /api/v1/city`

## Принимаемые заголовки

| Название           | Значение             | Обязательность |
|--------------------|----------------------|----------------|
| Content-Type       | application/json     | +              |
| Authorization      | Bearer auth_token    | +              |


Пример ответа
-------------

```json
[
    {
        "id": 2,
        "name": "Киев"
    },
    {
        "id": 3,
        "name": "Одесса"
    },
    {
        "id": 1,
        "name": "Харьков"
    }   
]
```
